# autosuspend-fork.koplugin
This is an alternative to the default autosuspend plugin with some changes to make the disable function works properly on kindles

I've tested it only in my kindle paperwhite 10th gen, and it seems to be working well.
**BACK UP YOUR ORIGINAL AUTOSUSPEND FILES BEFORE INSTALLING!**

### Installation
You should replace the original autosuspend.koplugin files with the files in this repository, and after that restart koreader.
